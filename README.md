# Cudanews Web Server Repo

## Install and run locally

**Clone the repo**

```bash
$ git clone https://github.com/Shane325/cudanews-webserver.git
```

**Install dependencies**

```bash
$ cd /cudanews-webserver
$ npm install
```

**Run app locally**

```bash
$ node bootstrap.js
```

The app will now be running on port 8000.
